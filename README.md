# cursoIonic

## ¿Que es?

Framework basado en Angular que permite crear aplicaciones hibridas 

## Contenido

1. Introducción y setup del entorno
2. Aspectos básicos de Ionic
3. Navegación
4. Temas(estilo)
5. CRUD en ionic y Firebase
6. Deployment
7. Conclusión


## Prerequisitos

* Javascript
* Html
* Css
* Api Rest
* Adicionales
  - Typescript
  - Angular
  - Desarrollo movil en cualquier plataforma
  
  ## Herramientas usadas
  * Navegador web
  * Editor de texto
  * Node JS (*npm*)
  * Ionic CLI
  
  ## Ionic CLI
  
  Ionic CLI nos permite:
  * Crear proyectos 
  * Generar
    - Componentes
    - Rutas
    - Servicios
    - Pipes
    - Etc
  * Correr app localmente
  * Hacer testing
